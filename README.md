# Финансовый планер

## Описание проекта
Python-приложение для учета доходов и расходов с анализом и визуализацией данных

## Функционал
- Добавление дохода и/или расхода;
- Данные хранятся в CSV;
- Анализ расходов;
- Наглядный просмотр данных;
- GUI Tk.

## Запуск
```bash
pip install -r requirements.txt
python main.app
```
Итоговый проект по курсу Программная инженерия. Разработка ПО (Python для начинающих специалистов)
